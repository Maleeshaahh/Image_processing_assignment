# Football Player Tracking System using Image Processing

This project implements a football player tracking system using image processing techniques. The system processes a football match video and tracks a selected player across frames using computer vision algorithms.

The project focuses on three main stages:

1. Image Enhancement
2. Player Segmentation
3. Player Tracking

The implementation is done using Python and OpenCV.

# Project Objectives

- Improve the quality of video frames using image enhancement techniques.
- Isolate the football player from the background using segmentation methods.
- Track the selected player across multiple frames in the video.
  
# Technologies Used

- Python
- OpenCV
- NumPy

# Project Pipeline

The system follows the following processing pipeline:

1. Video Input  
   - A football match video is used as the input.

2. Frame Extraction  
   - The video is converted into individual frames.

3. Image Enhancement  
   - Frames are enhanced to improve visibility and quality.

4. Player Segmentation  
   - The player is separated from the background using segmentation techniques.

5. Player Tracking  
   - A tracking algorithm is used to follow the selected player across frames.
